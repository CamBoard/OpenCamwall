# OpenCamwall

继承自个人闭源项目camwall-backend  
实现从接收投稿、稿件审核到发表说说的自动化

## 依赖

### 库

* flask
* requests
* Pillow
* pymysql
* yiri-mirai

### 框架

* [Mirai](https://github.com/mamoe/mirai) QQ机器人框架
* [YiriMirai](https://github.com/YiriMiraiProject/YiriMirai) Mirai框架Python SDK
